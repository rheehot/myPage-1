div and span

﻿div: 한 줄에 하나의 블록을 표시 및 꾸며줌
span: 컨텐츠를 표시 및 꾸며줌

display:

block: 한줄에 하나를 표현해주는 상자
inline-block: 한 줄에 여러개가 표현가능한 상자
inline: 컨텐츠만을 표현
﻿*div의 기본 디스플레이: block
*span의 기본 디스플레이: inline

position:

static: 기본
relative: 기본 자리를 기준으로
absolute: 상자 모서리를 기준으로
fixed: 상자 밖 페이지 모서리를 기준으로
sticky: 스크롤해도 기본 자리 유지

git

VCS(Version Control System)의 한 종류로, 버전에 대한 변화들을 관리해주는 시스템을 뜻한다.

git 기본 사용법

1. git 디렉토리 탐색
2. git 명령어 사용!
*기본 명령어
-git log: 기록을 표시
-git status: 상태를 표시
-git add -A: 현재까지의 변경사항을 커밋할수 있도록 준비
-git commit -m "title": add된 현재까지의 변경사항을 title이라는 이름으로 저장
-git reset ****** --hard: ******의 상태로 백업
-git revert ****** + (:wq): ******의 상태로 새롭게 커밋
